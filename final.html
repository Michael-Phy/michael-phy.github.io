Three Project Options

1. Parkinson’s Anti-Tremor Tableware Helper

This wearable arm tool includes a custom printed frame, soft padding, a simple hand sensor, a small motor, regular lights, and a reminder buzzer. The soft material reduces shaky hand movements while people eat. If the user cannot hold their utensils steadily, the sensor will activate the motor to hold chopsticks or spoons in place and prevent slipping. The lights turn on automatically in dark rooms, and the buzzer gives soft reminder sounds. This project creates a cheap and useful daily tool. It uses basic making skills to help people with hand movement problems and make simple assistive tools easier to access.

2. Wearable Reading Magnifier for Poor Eyesight

This lightweight head-mounted device is built with a custom printed frame, a light sensor, simple buttons, adjustable lights, and a small moving part for angle control. It changes light brightness automatically based on the room environment, and users can adjust the lens angle to see clearer. This design helps people with blurry eyesight read books and screens more comfortably. Instead of using expensive store-bought devices, this project uses simple handmade skills to create a low-cost option. It shows how easy homemade technology can help people with poor eyesight in daily life.

3. Posture Correction Wearable Device

This wearable back device is made of a comfortable printed frame, a position sensor, a soft vibration piece, and small indicator lights. The sensor keeps checking the user’s sitting posture all the time. When the user slouches or leans forward too much, the device vibrates softly to remind them to sit straight. It works very quietly and makes no loud noise at night. This project fixes common body problems caused by long screen time and online studying. It uses simple making techniques to help students build better daily habits and improve physical health from digital device overuse.

My Preferred Project

Among all three ideas, my top choice is the Parkinson’s Anti-Tremor Tableware Helper, as it can directly improve the dining independence of people living with Parkinson’s disease.
